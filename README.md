# 📊 A 股投资看板

基于真实 A 股分析报告构建的金融科技级暗黑复盘仪表盘。

## 功能

- **📈 核心指数矩阵** — 上证/深证/创业板/科创50/上证50 实时行情卡片
- **📊 大盘总量指标** — 成交额、上涨个股等市场流动数据
- **🧠 市场情绪看板** — 量能/结构/赚钱效应/资金偏好/情绪热度
- **📰 海内外资讯** — 国内 + 全球市场动态与解读
- **💰 持股收益追踪** — 添加自选股，自动计算盈亏（localStorage 持久化）
- **📈 投资分析** — 板块配置饼图、组合风险评估、催化事件时间线

## 启动方式

```bash
node server.js
# 访问 http://localhost:3000
```

行情数据通过腾讯免费 API 实时获取，每 30 秒自动刷新。

## 技术栈

纯静态 HTML/CSS/JS + Node.js 本地代理服务器

## 数据来源

腾讯行情 API · 财联社 · 东方财富 · 新浪财经 · 同花顺
